It goes through the entire elements one by one and hence it’s not a very efficient algorithm to work on large data sets. 1. How does Selection sort work? Selection sort starts with an unsorted data set. With each iteration, it builds up a sub dataset with the sorted data. By the end of the sorting process, the sub dataset contains the entire elements in a sorted order.
